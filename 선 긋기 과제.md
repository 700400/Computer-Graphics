# 선 긋기 과제
```
1. 교수님의 도움으로 쉽게 설치하고 바로가기 아이콘을 만들어냈습니다.

2. 프로그램을 코딩해서 결과를 실행해봤습니다. 생각보다 선이 굵다고 느꼈습니다.

3.스마트폰에서 실행 성공했습니다. 연결되지않은 각각의 기기에서 같은 프로그램이 실행되는게 신기했습니다.

1) 자바스크립트 모드 추가완료했습니다.

2) 초록색이 아닌 파란색으로 굵기 20으로 수정해봤습니다.

4. 1학년을 마치고 군대를 가게되어서 아무것도 모르는 상태였지만 심재창 교수님의 열정적인 강의 덕분에 빠르게 컴퓨터 그래픽스를 마스터 할 수 있을 것 같습니다.
이에 만족하지않고 1학년때 배웠던 c언어를 더 공부해서 잊었던 내용을 다시 공부하겠습니다.  다른 컴퓨터 언어도 추가로 공부해서 컴퓨터 언어의 마스터가 되겠습니다.
컴퓨터 그래픽스 처음 배워봤는데 교수님께서 너무 재밌고 쉽게 설명해주셔서 더욱 흥미가 생깁니다. 감사합니다! 앞으로도 열심히 배우겠습니다.
```
