# 3차원 지형의 생성
```java
int cols, rows;
int scl = 20; //scale
int w = 2000;
int h = 2000;

float flying = 0;

float[][] terrain;

void setup()
{
  size(600,600,P3D);

  cols = w / scl;
  rows = h / scl;
  terrain = new float[cols][rows];

}
void draw()
{
  flying += 0.1;
  
    float yoff = flying;
  for(int y = 0; y < rows-1; y++){
    float xoff = 0;
    for(int x = 0; x < cols; x++) {
      terrain[x][y] = map(noise(xoff,yoff),0,1,-130,130);
      xoff +=0.2;

    }
    yoff += 0.2;
  }
  
  background(0);



  translate(width/2,height/2+50);
  rotateX(PI/3);
  
  translate(-w/2,-h/2);
  for(int y = 0; y < rows-1; y++){ // y=0, x= 0~rows-1
    beginShape(TRIANGLE_STRIP);
    for(int x = 0; x < cols; x++){
      vertex(x*scl,y*scl, terrain[x][y]);
      vertex(x*scl,(y+1)*scl, terrain[x][y+1]);
      //rect(x*scl,y*scl,scl,scl);
        fill(x,y,x-y);
        stroke(0);
    }
    endShape();
  }
}

```

# 변형점

> * 점점 앞으로 나아가는 방향이 아닌 뒤로 이동하는 방향으로 fluing의 값을 +0.1으로 수정해봤습니다.
> * 저는 map이 noise의 0~1이었던 범위를 재지정을 한다는 것을 공부한뒤 -100,100이었던 값을 변동폭이 더 큰 -130,130으로 바꿔봤습니다.
> * fill을 이용해서 지형이 x와 y값을 받아서 위치에 따라 점점 바뀌게 만들어봤습니다.
